# 阿柑少年网站架构与品牌定位改版规划

Date: 2026-06-18
Status: meeting-calibrated planning document

## 1. 改版目标

这次改版不只是把网站做漂亮，而是把阿柑少年的品牌、内容和用户路径重新校准成一个更清晰的系统：

> 让第一次访问网站的人，在 30 秒内理解阿柑少年是什么、为什么独特、如何继续了解或加入。

网站应从「内容堆叠型介绍页」升级为「简洁有力的品牌吸引页 + 分工明确的内页系统」。

核心方向：
- 首页变轻：只负责吸引、定位、情绪冲击和入口。
- 内页变清楚：About 讲来处，Action 讲正在做什么，Join 讲如何加入和社群声音。
- 视觉变强：用真实素材和滚动叙事呈现山野、田野、城乡行动。
- 内核变硬：用三层行动逻辑承接整全生命成长，而不是只讲抽象理念。

## 2. 会议决策吸收

本规划以会议纪要为最高优先级输入，对此前研究文档做如下校准：

| 方向 | 会议决策 | 规划调整 |
| --- | --- | --- |
| 首页 | 简洁有力的吸引页 | 首页不再承载完整项目详情、长历程、复杂转化说明 |
| 品牌定位 | 真实社区中的整全生命成长 | 从“生态行动平台”升级为“真实社区中的整全生命成长计划” |
| 行动体系 | 山野-田野-城乡三层逻辑 | 作为 Action 页和首页动效的主结构 |
| 内页 | About + Journey 合并，Field + Action 合并 | 保留 `/about`、`/actions`、`/join` 三个主内页 |
| 社群 | 种子社群、伙伴之声 | Join 页增加声音/播客/感悟模块 |
| 素材 | 整理图片、账号、域名邮箱 | 新增素材收集清单与实施前置任务 |

## 3. 品牌定位

### 3.1 主品牌句

中文：

> 阿柑少年，是在真实社区中的整全生命成长计划。

英文建议：

> R'gan Junior is a whole-person growth journey in real communities.

### 3.2 首页短表达

中文：

> 在山野恢复感知，在田野理解真实问题，在城乡行动中长成自己。

英文建议：

> Restore your senses in the wild, understand real problems in the field, and grow through urban-rural action.

### 3.3 品牌关键词

- 真实社区
- 整全生命成长
- 山野感知
- 田野研究
- 城乡行动
- 种子社群
- 长期陪伴
- 跨学科真实连接

### 3.4 不要成为

- 普通研学营
- 户外夏令营
- 公益活动展示站
- 创始人履历页
- 只靠吉祥物和可爱感的青少年品牌

## 4. 三层行动逻辑

三层行动逻辑是新版网站的核心骨架。

### 4.1 山野探索：基础层

定位：

> 回到原始自然，修复被城市生活钝化的感知系统。

核心价值：
- 身心修复
- 五感重建
- 自然经验
- 生态整体感
- 为后续田野行动储备能量

可展示素材：
- 森林、山路、溪流、露营、徒步
- 少年观察、书写、安静独处、集体同行
- 身体感知、自然疗愈、夜谈或围坐

首页表达：

> 先回到山野，重新听见身体和自然。

### 4.2 田野调查：中间层

定位：

> 进入真实乡村社区，把自然感知转化为问题意识和研究能力。

核心价值：
- 铁牛村真实社区
- CSA 经济学调研
- 家庭访谈
- 土壤修复与生态农业
- 行为经济学、社会学、生态学的跨学科理解

可展示素材：
- 访谈、问卷、田地观察、果园劳动
- CSA 产品、生态盲盒、调研表、手写笔记
- 铁牛村地图、林盘、农户、土壤细节

首页表达：

> 再进入田野，把感受变成理解真实世界的能力。

### 4.3 城乡行动与山野互动：应用层

定位：

> 连接城市与乡村，把调研、表达和社群行动转化为真实影响。

核心价值：
- 城乡联动
- 校园 CSA
- 青少年发声
- 土壤改良行动
- 传播倡导
- 社群共建

可展示素材：
- 校园 CSA 活动
- 论坛展示、发表、公众表达
- 社群行动现场
- 城市家庭与乡村生产者的连接

首页表达：

> 最后回到城乡行动，让成长进入真实社会关系。

## 5. 网站信息架构

### 5.1 顶层导航

建议保留四个入口：

- 首页 Home
- 关于 About
- 行动 Action
- 加入 Join

不再增加复杂导航。旧路径继续重定向：
- `/journey` → `/about`
- `/field-research` → `/actions`
- `/voices` → `/join#voices`

### 5.2 首页职责

首页只做五件事：

1. 建立第一眼冲击：阿柑少年是什么。
2. 讲清核心信念：为什么真实社区和整全生命成长重要。
3. 露出三层行动：山野、田野、城乡。
4. 呈现社群召唤：不是一个人长大，而是一群种子少年共同前行。
5. 给出下一步入口：了解行动 / 加入我们。

首页不做：
- 长项目发展史
- 过多创始人背景
- 完整家长 FAQ
- 大量成果堆叠
- 所有行动细节

### 5.3 About 页职责

About 页整合：
- Nate 团队背景
- 铁牛村起源
- 阿柑少年从 1.0 到 3.0 的发展历程
- 为什么铁牛村不是背景，而是教育现场
- 团队和项目可信度

About 页语气应更像“来处与根基”，而不是简历墙。

### 5.4 Action 页职责

Action 页整合原“田野活动”和“行动记录”，用三层行动逻辑组织：

1. 山野探索
2. 田野调查
3. 城乡行动与山野互动

每一层都应包含：
- 这一层解决什么成长问题
- 当前对应项目
- 真实照片/视频
- 研究或行动证据
- 下一步如何参与

### 5.5 Join 页职责

Join 页负责转化和社群信任：
- 成为阿柑少年
- 成为阿柑家长
- 成为合作伙伴
- 伙伴之声
- 当前阶段说明
- 联系方式

伙伴之声可以先用文字卡片占位，后续升级为音频/播客。

## 6. 首页新版结构

### Section 1: Hero

目标：

> 5 秒内给出品牌定位和视觉冲击。

内容：
- 阿柑少年
- 在真实社区中的整全生命成长计划
- 在山野恢复感知，在田野理解真实问题，在城乡行动中长成自己
- CTA: 了解我们的行动 / 加入我们

视觉：
- 优先使用真实山野/田野/少年素材作为主视觉。
- 吉祥物从主角降级为品牌签名或动效陪伴。
- 首屏可使用滚动视频感：一组图片缓慢切换或视差推进。

### Section 2: 我们相信什么

目标：

> 用极少文字讲清底层价值。

建议三句：
- 自然是最好的老师。
- 真实世界是最深刻的课堂。
- 青少年不是未来，而是正在发生的力量。

这一段保留诗性，但不要过长。

### Section 3: 三层行动逻辑

目标：

> 首页的核心结构和视觉高潮。

表现方式：
- 三段横向或纵向滚动叙事。
- 山野、田野、城乡各对应一张主图、一个动词、一句解释。
- 滚动时图片和文字切换，形成“从自然感知到社会行动”的推进。

文案示例：
- 山野探索：修复感知，恢复生命能量。
- 田野调查：进入真实问题，学习跨学科理解。
- 城乡行动：连接社群，把理解转化为行动。

### Section 4: 种子社群

目标：

> 回应会议中“长期坚守、相互支持、对抗孤立感”的社群价值。

内容：
- 阿柑少年不是一次活动，而是一群认知相近的少年长期同行。
- 在共同探索、共同研究、共同行动中，建立真实的支持关系。

视觉：
- 群体影像
- 网络线条动效可保留，但要更克制，像“同行关系”而不是科技网络。

### Section 5: 加入入口

目标：

> 简单、明确、有行动。

入口：
- 我是青少年
- 我是家长
- 我是合作伙伴

每个入口一句话，不展开长文。

## 7. 视觉与动效方向

### 7.1 视觉基调

方向：

> 野性真实 + 田野编辑感 + 少年生命力。

保留：
- 纸张底色
- 森林绿
- 阿柑橙
- 宋/明风格标题

增强：
- 更强的真实照片
- 更少装饰性卡片
- 更多全宽影像段
- 更明确的图片层级
- 更克制、更高级的动效

### 7.2 首页动画方案

推荐采用“三层行动滚动叙事动画”：

1. 用户进入首页，看到真实场景 hero。
2. 下滑后进入三层行动 section。
3. 山野图像先占据画面，文字解释感知重建。
4. 滚动进入田野，图像切换到铁牛村、访谈、CSA。
5. 再进入城乡行动，图像切换到校园、论坛、社群行动。
6. 最后落到 Join CTA。

技术上可用现有 Framer Motion + IntersectionObserver / scroll progress 完成，不必引入新框架。

### 7.3 素材优先级

必须收集：
- 山野探索素材：森林、山路、徒步、自然观察、少年独处和群体同行
- 田野调查素材：铁牛村、果园、农户访谈、调研表、CSA 产品、土壤
- 城乡行动素材：校园 CSA、展示、演讲、论坛、社群行动
- 伙伴之声素材：参与者文字、音频、家长反馈、合作方评价
- 品牌素材：吉祥物、logo、favicon、OG 分享图

## 9. SEO 与转化策略

虽然首页要简，但 SEO 和转化基础必须补齐。

### 9.1 SEO 基建

优先做：
- 修复 `/sitemap.xml`
- 增加 canonical
- 增加 Open Graph image
- 首页、About、Action、Join 独立 title 和 description
- 增加 Organization / EducationalOrganization JSON-LD
- 为 Join 页 FAQ 预留 FAQPage schema

### 9.2 转化基建

优先做：
- 联系方式不再长期显示 Coming Soon
- Join 页保留邮箱，并尽快补微信或表单
- 加入入口按青少年、家长、合作伙伴分流
- 当前阶段说明写清楚：小规模深度探索、名额有限、通过官方渠道联系

## 11. 验证清单

- 首页 30 秒内能说清阿柑少年是什么。
- 首页不再重复 About / Action 的长内容。
- 三层行动逻辑清楚、递进、有画面。
- 伙伴之声有入口，即使音频后续补。

- 页面不显得普通研学或公益模板。

## 12. 待确认问题

1. 英文主品牌继续使用 `R'gan Junior`，还是对外改为 `R'gan Youth`？
2. 首页主 CTA 是“了解我们的行动”，还是“加入我们”，还是两者并列？
3. 山野探索是否已有明确地点名称和可公开素材？
4. Join 页是否可以公开微信、表单或具体联系人？
5. 伙伴之声第一批先用文字，还是直接采集音频？
6. 当前项目是否需要显示时间、名额、费用或招募状态？

## 13. 推荐首轮实施切片

首轮不要全站大拆，建议先做：

1. 更新品牌定位文案。
2. 首页改成轻量吸引页。
3. 新增三层行动滚动叙事模块。
4. Action 页按三层逻辑重排。
5. Join 页增加伙伴之声占位和更清晰联系方式。
6. 修复 sitemap、canonical、OG 基础 SEO。

这 6 件事完成后，网站会从“已经有内容的网站”变成“有清晰品牌结构和强第一印象的网站”。
