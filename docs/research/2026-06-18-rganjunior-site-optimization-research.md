# 阿柑少年网站优化研究与下一步行动方案

Date: 2026-06-18
Status: research integrated into next actions

## 0. 一句话结论

阿柑少年网站下一步不应继续堆内容，而应围绕一个清晰系统重构：

> 阿柑少年，是在真实社区中的整全生命成长计划。

网站结构应变成：

> 首页负责吸引与定位，About 负责来处与根基，Action 负责山野-田野-城乡三层行动，Join 负责加入与伙伴之声。

第一轮代码实施重点：

1. 更新品牌定位文案。
2. 首页改成简洁有力的品牌吸引页。
3. 用“山野探索 - 田野调查 - 城乡行动与山野互动”重构首页动效和 Action 页。
4. Join 页加入伙伴之声与更清楚的加入路径。
5. 同步修复 sitemap、canonical、OG 图等 SEO 基建。

## 1. 研究输入

### 1.1 外部 benchmark 输入

本轮参考了以下类型的网站：

- 顶级户外教育与青少年成长：Outward Bound、NOLS、Where There Be Dragons。
- 真实社区与生态教育：Green School Bali、Teton Science Schools、The Ecology School。
- 顶级视觉叙事与内容组织：Emergence Magazine、Patagonia Stories、National Geographic、The Pudding、Distill。

关键启发：

- 顶级教育网站不是只讲理念，而是快速回答“我是谁、适合谁、为什么可信、下一步怎么做”。
- 顶级户外/成长项目必须给家长明确的安全、照护、项目路径和联系方式。
- 顶级视觉叙事不是堆好看照片，而是让每个视觉段承担一个叙事任务。
- 顶级内容结构会把地点变成教育方法，而不是背景素材。

### 1.2 内部会议输入

会议主题：

> 阿柑少年网站架构与品牌定位研讨会

会议校准后的核心决策：

- 首页定位为“简洁有力的吸引页”。
- 品牌定位为“真实社区中的整全生命成长”。
- 项目内核采用“山野 - 田野 - 城乡”三层行动逻辑。
- About 与 Journey 合并。
- Field Research 与 Actions 合并。
- Join 页新增“伙伴之声”。
- 素材、账号、域名邮箱需要进入执行清单。

## 2. 当前网站诊断

### 2.1 最大问题

当前网站已经有真实内容和不错的视觉基础，但信息没有形成强结构。

主要问题：

- 首页仍偏“内容展示”，没有足够聚焦成第一眼吸引页。
- 吉祥物很有记忆点，但真实少年、真实山野、真实田野行动没有成为首屏主角。
- About、Action、Join 的分工还可以更清楚。
- 三层行动逻辑还没有成为全站主骨架。
- 家长和合作方需要的信任信息散落，没有形成路径。
- 线上 SEO 基建薄弱：`/sitemap.xml` 返回首页 HTML，不是真 sitemap；静态 HTML 主要是 SPA shell；缺少 canonical、OG image、结构化数据等基础。

### 2.2 需要保留的优势

- 品牌名“阿柑少年”有记忆点。
- 吉祥物有辨识度，但应降级为品牌签名和互动陪伴，而不是占据全部注意力。
- 铁牛村、校园 CSA、CTB、YSA Journal、Claremont、哈佛交流等证据是真实资产。
- 现有视觉基调“纸张底色 + 森林绿 + 阿柑橙 + 宋/明标题”适合继续发展。
- 现有代码栈 Vite + React + Tailwind + Framer Motion 足够支持下一轮视觉与动效升级。

## 3. 品牌定位

### 3.1 主品牌句

中文：

> 阿柑少年，是在真实社区中的整全生命成长计划。

英文建议：

> R'gan Junior is a whole-person growth journey in real communities.

### 3.2 首页短表达

中文：

> 在山野恢复感知，在田野理解真实问题，在城乡行动中长成自己。

英文建议：

> Restore your senses in the wild, understand real problems in the field, and grow through urban-rural action.

### 3.3 品牌关键词

- 真实社区
- 整全生命成长
- 山野探索
- 田野调查
- 城乡行动
- 跨学科真实连接
- 种子社群
- 长期陪伴

### 3.4 避免的误区

阿柑少年不应被表达成：

- 普通研学营
- 户外夏令营
- 公益活动展示站
- 创始人履历页
- 只靠可爱吉祥物支撑的青少年品牌

## 4. 全站信息架构

### 4.1 顶层导航

保留四个入口：

- Home / 首页
- About / 关于
- Action / 行动
- Join / 加入

旧路径继续重定向：

- `/journey` -> `/about`
- `/field-research` -> `/actions`
- `/voices` -> `/join#voices`

### 4.2 首页职责

首页只做五件事：

1. 建立第一眼冲击：阿柑少年是什么。
2. 讲清核心信念：为什么真实社区和整全生命成长重要。
3. 露出三层行动：山野、田野、城乡。
4. 呈现种子社群：不是一个人长大，而是一群认知相近的少年长期同行。
5. 给出下一步入口：了解行动 / 加入我们。

首页不做：

- 长项目发展史
- 过多创始人背景
- 完整家长 FAQ
- 大量成果堆叠
- 所有行动细节

### 4.3 About 页职责

About 页整合：

- Nate 团队背景
- 铁牛村起源
- 阿柑少年从 1.0 到 3.0 的发展历程
- 为什么铁牛村不是背景，而是教育现场
- 团队和项目可信度

About 页语气应是“来处与根基”，不是简历墙。

### 4.4 Action 页职责

Action 页整合原“田野活动”和“行动记录”，用三层行动逻辑组织：

1. 山野探索
2. 田野调查
3. 城乡行动与山野互动

每一层都应包含：

- 这一层解决什么成长问题
- 当前对应项目
- 真实照片/视频
- 研究或行动证据
- 下一步如何参与

### 4.5 Join 页职责

Join 页负责转化和社群信任：

- 成为阿柑少年
- 成为阿柑家长
- 成为合作伙伴
- 伙伴之声
- 当前阶段说明
- 联系方式

伙伴之声可以先用文字卡片占位，后续升级为音频或播客。

## 5. 三层行动逻辑

### 5.1 山野探索：基础层

定位：

> 回到原始自然，修复被城市生活钝化的感知系统。

核心价值：

- 身心修复
- 五感重建
- 自然经验
- 生态整体感
- 为后续田野行动储备身心能量

首页表达：

> 先回到山野，重新听见身体和自然。

可用素材：

- 森林、山路、溪流、露营、徒步
- 少年观察、书写、安静独处、集体同行
- 身体感知、自然疗愈、夜谈或围坐

### 5.2 田野调查：中间层

定位：

> 进入真实乡村社区，把自然感知转化为问题意识和研究能力。

核心价值：

- 铁牛村真实社区
- CSA 经济学调研
- 家庭访谈
- 土壤修复与生态农业
- 行为经济学、社会学、生态学的跨学科理解

首页表达：

> 再进入田野，把感受变成理解真实世界的能力。

可用素材：

- 访谈、问卷、田地观察、果园劳动
- CSA 产品、生态盲盒、调研表、手写笔记
- 铁牛村地图、林盘、农户、土壤细节

### 5.3 城乡行动与山野互动：应用层

定位：

> 连接城市与乡村，把调研、表达和社群行动转化为真实影响。

核心价值：

- 城乡联动
- 校园 CSA
- 青少年发声
- 土壤改良行动
- 传播倡导
- 社群共建

首页表达：

> 最后回到城乡行动，让成长进入真实社会关系。

可用素材：

- 校园 CSA 活动
- 论坛展示、发表、公众表达
- 社群行动现场
- 城市家庭与乡村生产者的连接

## 6. 首页下一版结构

### Section 1: Hero

目标：

> 5 秒内给出品牌定位和视觉冲击。

内容：

- 阿柑少年
- 在真实社区中的整全生命成长计划
- 在山野恢复感知，在田野理解真实问题，在城乡行动中长成自己
- CTA: 了解我们的行动 / 加入我们

视觉：

- 优先使用真实山野、田野、少年素材作为主视觉。
- 吉祥物从主角降级为品牌签名或动效陪伴。
- 首屏可做“滚动视频感”：一组图片缓慢切换或视差推进。

### Section 2: 我们相信什么

目标：

> 用极少文字讲清底层价值。

建议三句：

- 自然是最好的老师。
- 真实世界是最深刻的课堂。
- 青少年不是未来，而是正在发生的力量。

这一段保留诗性，但不要过长。

### Section 3: 三层行动逻辑

目标：

> 首页的核心结构和视觉高潮。

表现方式：

- 三段横向或纵向滚动叙事。
- 山野、田野、城乡各对应一张主图、一个动词、一句解释。
- 滚动时图片和文字切换，形成“从自然感知到社会行动”的推进。

文案示例：

- 山野探索：修复感知，恢复生命能量。
- 田野调查：进入真实问题，学习跨学科理解。
- 城乡行动：连接社群，把理解转化为行动。

### Section 4: 种子社群

目标：

> 回应“长期坚守、相互支持、对抗孤立感”的社群价值。

内容：

- 阿柑少年不是一次活动，而是一群认知相近的少年长期同行。
- 在共同探索、共同研究、共同行动中，建立真实的支持关系。

视觉：

- 群体影像。
- 现有 NetworkAnimation 可保留，但要更克制，表达“同行关系”，不是科技感网络。

### Section 5: 加入入口

目标：

> 简单、明确、有行动。

入口：

- 我是青少年
- 我是家长
- 我是合作伙伴

每个入口一句话，不展开长文。

## 7. 视觉与动效方向

### 7.1 视觉基调

方向：

> 野性真实 + 田野编辑感 + 少年生命力。

保留：

- 纸张底色
- 森林绿
- 阿柑橙
- 宋/明风格标题

增强：

- 更强的真实照片
- 更少装饰性卡片
- 更多全宽影像段
- 更明确的图片层级
- 更克制、更高级的动效

### 7.2 首页动画方案

推荐采用“三层行动滚动叙事动画”：

1. 用户进入首页，看到真实场景 hero。
2. 下滑进入三层行动 section。
3. 山野图像先占据画面，文字解释感知重建。
4. 滚动进入田野，图像切换到铁牛村、访谈、CSA。
5. 再进入城乡行动，图像切换到校园、论坛、社群行动。
6. 最后落到 Join CTA。

技术上用现有 Framer Motion + IntersectionObserver / scroll progress 即可，不必引入新框架。

3. 山野探索是否已有明确地点名称和可公开素材？
4. Join 页是否可以公开微信、表单或具体联系人？
5. 伙伴之声第一批先用文字，还是直接采集音频？
6. 当前项目是否需要显示时间、名额、费用或招募状态？

## 14. 推荐实施顺序

首轮不要全站大拆，建议按这个顺序：

1. 更新品牌定位文案。
2. 首页改成轻量吸引页。
3. 新增三层行动滚动叙事模块。
4. Action 页按三层逻辑重排。
5. Join 页增加伙伴之声占位和更清晰联系方式。
6. 修复 sitemap、canonical、OG 基础 SEO。

完成这 6 件事后，网站会从“已经有内容的网站”变成“有清晰品牌结构和强第一印象的网站”。
